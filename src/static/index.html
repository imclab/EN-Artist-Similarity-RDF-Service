1. Description

This service provides an RDF wrapper around the Echonest API for artist similarity. Artist URIs are constructed from the corresponding MusicBrainz identifier and follow the form: 

http://dbtune.org/artists/echonest/mbid/<mbid>

The returned RDF contains a series of similarities to other artists as specified by Echonest. All similarities are described using the MuSim Similarity Ontology.

Some example music artist resources include:

  • José James
  • James Brown
